
## Development

Install dependencies

```bash
pnpm install
```

Run build watcher

```bash
pnpm run dev
```

Build sdk

```bash
pnpm run build
```

Generate documentation

```bash
pnpm run docs
```

## Test

```bash
pnpm test
```
